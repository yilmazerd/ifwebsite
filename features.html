Using instant function, you can create a simple JSON response that represents production response from one of the API's that you need to call. You can simulate a variety of scenarios in the response. As an example, suppose you are developing a financial application and you call an api to get credit score of an individual. Suppose the respoinse of that 3rd party API has the following structure:

{ 
  "creditScore": "400", 
  "Report update date": "2020-10-10" 
}

Suppose you need to simulate cases to test against credit score scenarios. You can write a simple random generator and use instant function to create random responses to test your code.

Following up our finance application, you can simulate cases where the response is not 200 but 40x or 50x. For instance if you are expecting 95% reliability, you can create a random 5% error and when that happens return the erroneous response.

Especially with optimization tasks, it can be difficult to have a good estimate for the initial variables. Once the optimization code is defined then you can define initial variables in the request body to instant function and run instant function in a parallel form against these varitey of initializations. Current Functionality and Limitations 

As of today, Instant Function is Java only. I'm going to add Python next.


Execution
Basic execution of Java programs are around feature "Launch Single-File Source-Code Programs" introduced in JDK 11 release. Meaning you can test your code before submitting by typing "Java <myCodeName.java>" in your terminal. This limits the libraries that can be imported but since our scope is to run simple functinoality in the API, that has not been an issue. Let me know if your use case requires a library that is beyond single file source code functionality. I can take a look.

Response Code
Essentially the program will return what is being logged. You can customize the http response code. In order to do this, you need to print the following statement
System.out.println("httpStatusCode:"+httpStatusCode);
if this is not printed, then it will return 200.. I plan to keep this structure for python and other platforms. Shoot an email if you have a better idea..


Response Structure
If you customize the response code then we look for a log statement in the following structure
System.out.println("responseObject:"+responseObject);
once found, we return response object as JSON (at least we try). Response format is JSON and currently can not be edited. If you have a use case where you need to return non-json, shoot an email with your use case.
