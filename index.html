Welcome to Instant Function

Instant Function helps you to test your ideas and develop your code easily. Enter your code and your api is hosted by instant function and is immediately ready to use. Instant Function manages everything so you can focus on your ideas.

Get Started

How it works

Create Your Function

Step 1:
Select your platform (Java, Python, etc. )and enter your code. Follow instructions regarding how to handle requests and construct response or use one of the examples to start with

Step 2:
Submit your code. Instant function will provide you a public url. Start using your new API right away.

Start From One of the Examples

Use one of the examples to develop your ideas faster.

Things you can do with instant function

Use Instant Function as A Mock Service

Easily deploy dynamic mocks. Use it as your back end either even if it's an idea for now. Simulate various cases such as adding a variety of response types, http status codes or simulating server errors and latencies.

Develop Tools and Share with Others

As you develop your function, your api will become public. You can share your function with team mates to help others more effectively develop software. Develop tools such as text formatters, automatic code generators, json generators and share it with your team.

Use Existing Functions that others developed and shared with the community.
